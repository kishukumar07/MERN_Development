Data Encapsulation - Interview Notes

Q1: What is Data Encapsulation?

Answer: Data Encapsulation is the process of bundling data (variables) and methods that operate on the data into a single unit (like an object). It helps in restricting direct access to some components.

Q2: Why is Encapsulation important?
  • ‚úî Protects data from accidental modification
  • ‚úî Hides internal implementation
  • ‚úî Improves code maintainability
  • ‚úî Provides controlled access using getters & setters
Q3: How is Encapsulation achieved in JavaScript?

Encapsulation can be implemented using:

  • ‚úî Closures
  • ‚úî Private properties (ES6 #)
  • ‚úî Getters & Setters
Q4: Example of Encapsulation using Closures?

Closures allow us to hide variables inside a function, preventing direct access.

function createCounter() { let count = 0; // Private variable return { increment: function () { count++; console.log(count); }, decrement: function () { count--; console.log(count); }, getCount: function () { return count; } }; } const counter = createCounter(); counter.increment(); // Output: 1 counter.increment(); // Output: 2 console.log(counter.getCount()); // Output: 2 console.log(counter.count); // Undefined (Cannot access private variable)
Q5: Example using ES6 Private Fields?

With ES6, we can use the # symbol to create truly private properties inside classes.

class Counter { #count = 0; // Private field increment() { this.#count++; console.log(this.#count); } getCount() { return this.#count; } } const counter = new Counter(); counter.increment(); // Output: 1 console.log(counter.getCount()); // Output: 1 console.log(counter.#count); // ‚ùå Syntax Error: Private field cannot be accessed
Q6: Example using Getters & Setters?

Getters and setters allow controlled access to private properties.

class Person { constructor(name, age) { this._name = name; // Convention: _name means "private" this._age = age; } get age() { return this._age; } set age(newAge) { if (newAge > 0) this._age = newAge; else console.log("Age must be positive!"); } } const person = new Person("Niket", 20); console.log(person.age); // Output: 20 person.age = 25; // ‚úÖ Allowed (through setter) person.age = -5; // ‚ùå Age must be positive!
